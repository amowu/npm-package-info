# npm-package-info

Know project's npm packages easily.

## shell script version

- httpie (https://github.com/jkbrzt/httpie)
- jq (https://stedolan.github.io/jq/)

### Usage

```bash
./npi.sh https://github.com/FreeCodeCamp/FreeCodeCamp
```
